# Quick Start Guide

## Step-by-Step Setup

### 1. Generate Icons (1 minute)

1. Open `icon-generator.html` in your browser (double-click the file)
2. Click "Generate Icons" button
3. Right-click each canvas image and select "Save image as..."
4. Save them with exact names in the `icons/` folder:
   - icon16.png
   - icon48.png
   - icon128.png

### 2. Load Extension in Chrome (1 minute)

1. Open Chrome and go to: `chrome://extensions/`
2. Toggle ON "Developer mode" (top-right corner)
3. Click "Load unpacked" button
4. Select the `time-tracker-extension` folder
5. Done! The extension is now installed

### 3. Start Using (1 minute)

1. Click the extension icon in Chrome toolbar (puzzle piece ‚Üí Skill Time Tracker)
2. Create your first task:
   - Task name: "Learn Python"
   - Technique: Pomodoro
   - Click "Add Task"
3. Click "Start" on your task
4. Click the Start button in the timer
5. You're tracking time! üéâ

## Troubleshooting

**Can't see the extension icon?**

- Click the puzzle piece in Chrome toolbar
- Pin "Skill Time Tracker" for easy access

**Timer not working?**

- Allow notifications when prompted
- Refresh the extension at chrome://extensions/

**Want to reset data?**

- Go to chrome://extensions/
- Click "Remove" under Skill Time Tracker
- Reload the extension (data will be cleared)
